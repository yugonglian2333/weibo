1
点击下方按钮，浏览器会自动打开微博登录页
2
在浏览器中手动扫码或输入账号密码登录（最长等待 5 分钟）
3
登录成功后 Cookie 会显示在下方，输入名称后保存到 .env
需要 Playwright：pip install playwright && playwright install chromium
多账号用不同名称区分，如 WEIBO_COOKIE_A、WEIBO_COOKIE_B
